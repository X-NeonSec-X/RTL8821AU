# RTL8821AU

It is a bash script to install RTL8821AU driver in debian based Distros.

# Installation Steps:-

#1 :-

    sudo apt install wget -y 
   
#2 :-

    wget https://raw.githubusercontent.com/X-NeonSec-X/RTL8821AU/main/RTL8821AU
   
#3 :-

    sudo chmod +x RTL8821AU 
   
#4 :-

    sudo bash RTL8821AU
